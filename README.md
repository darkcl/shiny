Shiny Counter CLI
===

Usage
---

```shiny hunt "Xerneas"```

Start Hunting for Xerneas

```shiny hunt "Poipole Pikachu"```

Start Hunting for Poipole, Pikachu

```shiny count "Xerneas Poipole" for 1```

Add Counter for Xerneas and Poipole for 1

```shiny get "Xerneas"```

Complete shiny hunt for Xerneas
